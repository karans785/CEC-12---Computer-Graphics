OVERVIEW:

This program clips a polygon according to a clipping window using the Sutherland Hodgeman Algorithm.

INSTRUCTIONS: 

1.Kindly click on the .exe file (application)  polygon_clip.exe for viewing the output!
2.Firstly the clipping window will be visible.
3.Then the subject polygon appears on the screen.
4.To clip the polygon edge by edge ,press 'c' on your keyboard and watch the polygon getting clipped(basically changing color fro white to cyan)!!
