OVERVIEW:

This program clips a line segment according to a clipping window using the Liang-Barsky Algorithm.

INSTRUCTIONS: 

1.Kindly click on the .exe file (application)  liang.exe. 

2.Now enter the following input:
Clipping window : 2 3 3 1
Line segment      : 1 3 3 1
(The developer has scaled the input be a factor of 100 .So input accordingly!)

3.The window and line segment are drawn in RED color.

4.Now press 'c' on the keyboard as  C is for clipping :).

5. Watch the clipped line segment change its color.
